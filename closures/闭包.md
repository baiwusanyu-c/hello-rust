# Rust 闭包
Rust 的闭包是可以保存在一个变量中或作为参数传递给其他函数的匿名函数。
例如：
```rust
struct City {
    name: String,
    population: i64,
    country: String
}

fn sort_cities(cities: &mut Vec<City>) {
    cities.sort_by_key(city_population_descending);
}

/// 按照人口数量对城市进行排序的辅助函数
fn city_population_descending(city: &City) -> i64 {
    -city.population
}

```
